eWiz: Event-based Data Manipulation
===================================
Welcome to the official documentation for **eWiz**, a Python library for manipulating
and processing event-based data. Whether you're working with spiking neural networks,
event-based cameras, or synthetic datasets, eWiz provides the tools you need to
streamline your workflow.
